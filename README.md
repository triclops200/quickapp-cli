# quickapp-cli
Client for quickapp

This is a unix-style wrapper for quickapp so that one can create sbcl buildapp apps from the command line.

Example Usage:
```bash
 quickapp testproject \
   -d"(:sdl2 :cl-opengl)" \
   --author=cluser
```

See https://github.com/triclops200/quickapp for more options and examples.
